Distanciel de l'UE "Complexité et algorithmes", par Nicolas COMPÈRE et Adrien CALLICO

Pour lancer l'application sur une machine du CIE sous Linux:

Soit:

1/ double cliquer sur "main.py", puis cliquer sur "lancer dans un terminal"

ou

2/ Ouvrir un terminal, se placer dans le répertoire de main.py et exécuter la commande "python3 main.py" ou "python main.py"
selon la version de Python installée.
